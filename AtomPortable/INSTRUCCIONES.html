AtoM Portable

Access to Memory 2.10.0 + Asistente IA · Instalación local con Docker

Requisitos previos

🪟
Windows 10 / 11

64 bits. Cualquier edición.

🐳
Docker Desktop

Gratuito en docker.com. Versión 4.x o superior.

💾
Espacio en disco

Mínimo 8 GB libres (recomendado 16 GB).

🌐
Internet

Solo durante la instalación inicial (~3-4 GB de descargas).

Instalación inicial

Este proceso solo es necesario una vez. Tarda entre 20 y 30 minutos.

1
Instale y abra Docker Desktop

Espere a que el icono de la ballena en la barra de tareas deje de parpadear.

2
Coloque la carpeta AtomPortable donde desee

Puede instalarse en cualquier unidad: C:\AtomPortable, D:\AtomPortable, E:\AtomPortable... Los scripts detectan la ubicación automáticamente.

3
Ejecute 1_preparar_atom.bat

El script descarga AtoM desde GitHub, construye la imagen Docker, guarda todo para uso offline e inicializa la base de datos. No cierre la ventana.

⏱️ La construcción de la imagen puede tardar 10–20 minutos. Es normal.
4
Acceda a AtoM
http://localhost:8080
Usuarioadmin
ContraseñaAdmin2024!
🔑 Cambie la contraseña inmediatamente: menú superior → Admin → Mi perfil.

Uso diario

1
Abra Docker Desktop

Espere a que arranque completamente.

2
Ejecute 2_iniciar_atom.bat

Arranca AtoM y el Asistente IA simultáneamente.

3
Acceda a los servicios
AtoMhttp://localhost:8080
Asistente IAhttp://localhost:8081
⚠️ Importante: ejecute siempre 3_detener_atom.bat antes de apagar el equipo o desconectar el disco para evitar corrupción de datos.

Asistente IA

El asistente (http://localhost:8081) conecta con Ollama, un sistema de IA que funciona completamente en local. Ningún dato sale del equipo.

Instalar Ollama

Descargue e instale Ollama desde ollama.com. Luego descargue un modelo desde la línea de comandos:

ollama pull llama3.2

Usar el asistente

📋
ISAD(G)

Pegue texto descriptivo de un fondo, serie o documento. El asistente sugerirá los campos de la norma ISAD(G).

🏷️
Metadatos

Pegue el contenido de un documento (carta, informe, acta...). El asistente extraerá sus metadatos principales.

ℹ️ Configure la URL de Ollama y el modelo en la barra inferior del asistente. Los ajustes se guardan automáticamente.
